# JUN-s-Project

## Overview
This project is a modern, visually rich university portal demo. It features a high-end glassmorphism UI, a diverse mock student database, and a modular dashboard for student management and communication.

## Project Structure

```
JUN-s-Project/
│
├── index.html        # Main HTML file for the portal UI
├── style.css         # All visual styles, glassmorphism, and responsive layout
├── script.js         # Frontend logic, student data, dashboard, and messaging
├── campus.jpg        # (Optional) Background image for the hero section
└── README.md         # Project documentation
```

## Main Features
- Elegant landing page with glassmorphism and modern typography
- Student dashboard with sidebar navigation
- Diverse mock student data for demo purposes
- Messaging system (global and private)
- Responsive and accessible design

## Usage
Open `index.html` in your browser. The dashboard loads automatically for demo/testing. To restore authentication, re-enable the login logic in `script.js`.
